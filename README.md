# Rock Paper Scissors in C

A simple two-player Rock Paper Scissors game built in C. The program accepts player input, validates choices, determines the winner, and allows repeated play.

## Features
- Two-player console gameplay
- Accepts uppercase and lowercase input
- Rejects invalid choices and asks the user again
- Detects wins, losses, and ties
- Allows repeated rounds

## Skills Demonstrated
- C programming
- Enumerated types
- Functions and modular design
- Input validation
- Conditional logic and looping
